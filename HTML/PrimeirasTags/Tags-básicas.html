Caminho Servidor-Cliente ↑

O usuário envia uma requisição de IP para o DNS, que retorna o IP do site que o usuário pediu, assim o usuário pode acessar o site desejado. Isso é necessário porque o IP dos sites mudam periodicamente, o IP da sua internet também muda toda vez que o modem é tirado da tomada e reiniciado.

Teste do espaçamento de parágrafo e formatações ↑

Espaçamento de parágrafo

Para iniciar um parágrafo usa-se a tag <p>.
Para continuar o parágrafo na próxima linha usa-se a tag <br>

Negrito

Para usar negrito usa-se a tag <b> (Método sem semântica).
Porém, o correto seria usar o <strong> (Método com semântica).

Itálico

Para usar o itálico pode usar a tag <i> (Sem semântica).
Porém, deve-se usar a tag <em> (Com semântica).

Sublinhado

Para sublinhar um texto pode-se usar a tag <u> (Sem semântica).
Porém, deve-se utilizar a tag <ins> (Com semântica).

Texto excluído

Para esse tipo de formatação usa-se a tag <del>.
Frase de exemplo.

Marcação

Para adicionar marcação usa-se a tag <mark>. A cor padrão é amarela, mas é possível alterar utilizando o comando style na tag (já não é mais html, é css).

Texto sobrescrito

Para fazer o texto sobrescrito usa-se a tag <sup>
Exemplo de textosobrescrito, ou, x2+3.

Texto subscrito

Para fazer o texto subscrito usa-se a tag <sub>
Exemplo de textosubscrito , ou, H20.

Código-Fonte

A tag <code> deixa as letras monoespaçadas, com o mesmo espaçamento, facilitando a leitura em algumas ocasiões, como em links por exemplo.
Exemplo:
document.getElementByld'teste' (Sem o espaçamento).
document.getElementByld'teste' (Com o espaçamento).

Também é possível dentro do <code> com a tag <pre> formatar para que o texto assuma a mesma formatação usada no códig-fonte do site.
Exemplo: 

                    
                    #include <stdio.lib>
                    int main()
                    {
                        int N=2, N2=3, R;
                        R=N+N2;
                            printf("%d", R);
                    }

Citação Curta

Usa-se a tag <q>, para dar aspas e dar sentido a uma citação curta.
Ex: Como diria Tia Sônia: Receita original, fonte de fibras.

Citação Completa

Usa-se a tag <blockquote>, para fazer citações maiores (que possuem espaçamento).
Exemplo: Como diria Sun Tzu, no livro A Arte da Guerra:

A suprema arte da guerra é derrotar o inimigo sem lutar.
Também é possível adicionar o link de onde foi tirada a citação, mas serve apenas para que os créditos sejam devidamente atribuidos, não possui nenhuma funcionalidade no site.

Abreviações

Usa-se a tag <abbr> para colocar o significado de uma abreviação ao passar o mouse.
Ex: HTML, CSS.
A parte visual pode ser editado no css.

Inverter texto

Para inverter o texto usa-se a tag <bdo>, ela possui dois parâmetros, o ltr (left to right), e o rtl (right to left).
Ex: Tag bem aleatória / Tag bem aleatória.

Teste de adição de imagens ↑

Tag <img>

Imagem Adaptativa (sem CSS)

Para uma imagem adapatar ao tamanho do navegador ou da tela usa-se antes a tag <picture>, e a tag <source:media:type>.

Adicionando Áudio ↑

Para adicionar áudio usa-se a tag <audio>, seguida de alguns parâmetros de configurações como preload="", controls, autoplay, loop.

É possível colocar variáveis do formato do arquivo de áudio usando o <source:src>, em casos do navegador do usuário não suportar determinado tipo de arquivo de áudio.

Adicionando Videos ↑

Hospedados no Próprio Site

Para adicionar videos usas-se a tag <video>, seguida de alguns parâmetros, como controls (botões do vídeo), widht (tamanho que o video aparece) e/ou poster=" " (imagem da thumb).

É recomendado usar a tag <source:src> para adicionar vários tipos de arquivo de vídeo, para o caso do navegador do usário não ser compatível com algum.
Também é necesssário tomar cuidado com o tamanho do vídeo, o consumo para vídeos hospedados no próprio site é muito grande, aumentando os custos.

Hospedados de Forma Externa

São vídeos hospedados em outros sites, como o YouTube.
Ex:

O próprio YouTube tem o código pronto em HTML, basta clicar em compartilhar e em embed(incorporar). Lembrando que é necessário tomar cuidado com direitos autorais. Logo, se torma mais barato para o proprietário do site, postar os vídeos no YouTube e depois adicionar ao seu site de forma externa.

Lista com tag "ul" (tag sem ordenação) ↑

  • Item;
  • Item;
  • Item.

É possível alterar o formato do marcador usando type=" " podendo ser square, circle ou disc(padrão).

Lista com tag "ol" (tag com numeração)

  1. Item;
  2. Item;
  3. Item;

É possível alterar o início da contagem e o tipo de contador usando start=" " e type=" ". A contagem pode ser com numeros, com alfabeto(uppercase ou lowercase) ou em romano (uppercase ou lowercase)

Adicionando Emojis
